# 小车端对接文档

## 1. 通信方式

ROS2 Topic 双向通信，消息类型 `custom_msgs/msg/FiveFloats`。

## 2. 我方发布 → 小车接收

| 项目 | 内容 |
|------|------|
| **Topic** | `/topic_rk3588_to_a733` |
| **方向** | BCI 仪表盘 → 小车 (A733) |
| **消息类型** | `custom_msgs/msg/FiveFloats` |
| **QoS** | depth=10, 默认可靠传输 |

### FiveFloats 字段说明

| 字段 | 类型 | 发送内容 |
|------|------|----------|
| `attention` | float32 | **真实 BCI 注意力值** |
| `target_speed` | float32 | 0.0（预留） |
| `current_speed` | float32 | 0.0（预留） |
| `data4` | float32 | 0.0（预留） |
| `data5` | float32 | 0.0（预留） |

## 3. 注意力值语义（重要！）

```
attention 是反向指标：值越大 = 越放松，值越小 = 越专注

  0.0  ─────────  0.5  ─────────  1.0
  高度专注          中等          完全放松
  (紧张/警觉)                   (不专注/放空)
```

**底层原理：** 该值来自 BrainFlow ML 模型的 `RESTFULNESS`（放松度）输出，范围 0.0~1.0。
- 接近 0.0 → 大脑处于高度警觉/紧张状态
- 接近 1.0 → 大脑处于放松/休息状态

**典型用法（建议）：**
```python
effective_coefficient = 1.0 - attention
# attention=0.0 (专注) → coefficient=1.0 (全速)
# attention=1.0 (放松) → coefficient=0.0 (停止)
```

**发布频率：** 每次收到新的注意力数据时发送（BCI 数据流刷新频率约 30Hz）

## 4. 我方接收 ← 小车发送

| 项目 | 内容 |
|------|------|
| **Topic** | `/topic_a733_to_rk3588` |
| **方向** | 小车 (A733) → BCI 仪表盘 |
| **消息类型** | `custom_msgs/msg/FiveFloats` |

### 我方需要的字段

| 字段 | 类型 | 含义 | 用途 |
|------|------|------|------|
| `target_speed` | float32 | 小车的**线速度** (m/s) | 转发到云端 |
| `data4` | float32 | 小车的**角速度** (rad/s) | 转发到云端 |

其他字段（`attention`, `current_speed`, `data5`）我方不使用。
